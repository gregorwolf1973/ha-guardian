# Changelog

## 1.0.0 - 2026-03-28

### Added
- Initial release
- Automatic brute-force detection from Home Assistant logs
- IP banning via native `ip_bans.yaml` integration
- Configurable thresholds: max attempts, time window, ban duration
- Web UI with Dashboard, Blocked IPs, Whitelist, and Settings tabs
- Manual ban/unban via UI
- Whitelist support for individual IPs and CIDR ranges
- Trusted domains configuration
- Automatic ban expiry
- HA Ingress support
